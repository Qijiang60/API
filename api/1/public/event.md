##1/event/get/[eventid]
Fetches all contact information about the events

###Example
* <http://api.crew.dreamhack.se/1/event/get> -- Fetches all active events
* <http://api.crew.dreamhack.se/1/event/get/all> -- Fetches all events
* <http://api.crew.dreamhack.se/1/event/get/10184> -- Fetches a single event
